# 2DSurvival

2DSurvival is a web browser survival game where your goal is to defeat the boss.

[Press this to play the game](https://markussendzelis.github.io/2DSurvival/)

## Features
- Day/night cycle - world gets darker at night, campfires help you to survive
- Animals - different wild animals that will hunt you
- Crafting - spear which you can use to defeat animals or bosses
- Building - place and destroy wall to build or defend yourself
- Quests - quests that lead up to a boss fight
- Boss - a giant bear with 500hp that u have to defeat to win
- Minimap - shows your position, lakes and animals

## How to play
Controls:
- WASD - move
- E - chop tree / mine rock / pick up food / drink water
- F - eat food
- C - open crafting
- B - place campfire
- V - cycle between build / delete / normal mode
- Left click - attack animals / place walls while in build mode

Gather resources, craft, build stuff and survive long enough to complete all quests and defeat the boss.

## Tech stack
Built with Phaser and JavaScript
